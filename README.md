# ae3.sdk

The core L2 target-context framework and shared helpers that most of AE3 is built on — implements against the interfaces `ae3.api` declares. Requires `ae3.api`.

Beyond the core `ru/myx/ae3` framework code, it bundles several standalone helper packages: `auth`, `geo`, `jdbc`, `plist`, `query`, `renderer`, `sapi`, `time`, `util`, plus a benchmarking harness (`benchmarking`) and a file-watcher (`file_watcher`).
